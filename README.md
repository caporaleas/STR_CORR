# STR_CORR
Code to evaluate correlation between structural MRI metrics (specifically, R1 and SANDI diffusion parameters, but can be easily edited to accomodate for other metrics)

Please, if you use this code, cite: 
"Bliakharskaia, E., Chiarelli, A. M., Patitucci, E., Carriero, M., Di Censo, D., Biondetti, E., ... & Caporale, A. (2025). Exploring the contribution of gray matter microstructure to R1 contrast via multi-compartment diffusion modelling in the healthy brain. NeuroImage, 121466."
